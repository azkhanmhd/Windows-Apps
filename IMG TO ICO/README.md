# IMG TO ICO

A simple, Windows desktop application built to convert standard images (`JPG`, `JPEG`, `PNG`) into High Quality Icon `.ico` files.

## üöÄ Key Features

* **Custom Sizing:** Choose from multiple resolutions (16x16 up to 2048x2048).
* **Rounded Corners:** Built-in slider to add a professional corner radius to your icons.
* **Multi-Resolution Support:** The "Default" mode creates an icon containing multiple sizes for better scaling.
* **Simple UI:** Clean, distraction-free interface for quick conversions.

1. **Select Image:** Click "Browse" to choose your source image.
2. **Choose Size:** Select your desired output resolution from the dropdown menu.
